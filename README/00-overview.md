# Morning Command Dashboard

A personal dashboard that generates a daily action briefing from Gmail and Google Calendar.

## Goals

- Know what to build, who to contact, and what to pay every morning.
- Reduce context switching and avoid missed follow-ups.
- Keep implementation lightweight and maintainable for a solo founder.

## Inputs

- Gmail (primary)
- Google Calendar (primary)
- Optional future sources: Stripe, accounting, CRM, bank alerts

## Outputs

- Daily briefing at 07:00
- Prioritized actionable task list
- Dashboard panels: Develop, Customers, Finance, Calendar

## Non-goals (MVP)

- Full CRM replacement
- Complex workflow automation
- Team and multi-user support
